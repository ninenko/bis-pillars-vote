How are these numbers calculated?

When you rank the pillars, each position scores points: 1st choice = 4, 2nd = 3, 3rd = 2, 4th = 1. We add up everyone's points for each pillar. The number on each bar is the average points it received — 4.0 means everyone placed it first, 1.0 means everyone placed it last — and the bar length shows that score as a share of the maximum. The top block uses people's own rankings (Vote 1); the bottom uses their predictions of the team (Vote 2).
